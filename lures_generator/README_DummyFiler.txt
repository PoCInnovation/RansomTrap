
Dummy Filer GUI — Mode d'emploi rapide

1) Prérequis : Python 3 avec tkinter (inclus par défaut sur Windows/macOS; sur certaines distros Linux installez 'python3-tk').
2) Téléchargez le fichier 'dummy_filer_gui.py' et lancez :
   - Windows : double-cliquez (ou clic droit > Ouvrir avec Python)
   - Terminal : python dummy_filer_gui.py
3) Dans l'interface :
   - Choisissez le dossier racine où générer les fichiers.
   - Indiquez le nombre total de fichiers.
   - Cochez "Utiliser les sous-dossiers existants" OU indiquez combien de nouveaux sous-dossiers créer.
   - Choisissez les extensions à générer et la taille min/max (en KB).
   - Cliquez sur "Générer".
4) Le programme crée et répartit les fichiers automatiquement, avec des noms réalistes (factures, CV, etc.).
   Les .txt/.csv/.json peuvent contenir du texte factice lisible. Les autres sont remplis de données aléatoires.

Remarques : les fichiers créés sont purement factices (aucune donnée personnelle).
